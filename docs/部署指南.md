## 方案一：直接复制文件夹（推荐）

### 适用场景
- 给家人/朋友快速演示
- 临时在他人电脑使用
- 不需要网络连接

### 操作步骤

#### 1. 准备文件
```
将整个 aphasia-math-recovery 文件夹复制到 U 盘/微信文件传输/邮件附件
```

#### 2. 在目标电脑运行
**Windows:**
1. 复制文件夹到桌面
2. 双击打开 `index.html` 文件
3. 系统会自动用默认浏览器打开

**Mac:**
1. 复制文件夹到桌面
2. 右键 `index.html` → 打开方式 → Safari/Chrome

#### 3. 创建快捷方式（可选）
**Windows:**
1. 右键 `index.html` → 发送到 → 桌面快捷方式
2. 以后双击桌面图标即可打开

**Mac:**
1. 将 `index.html` 拖到 Dock 栏
2. 以后点击 Dock 图标即可打开

### 优点
- ✅ 无需安装任何软件
- ✅ 无需网络连接
- ✅ 数据保存在本机
- ✅ 操作简单

### 缺点
- ⚠️ 需要手动复制文件
- ⚠️ 不会自动更新
- ⚠️ 浏览器地址栏显示本地路径

## 方案二：打包成桌面应用

### 适用场景
- 正式交付给患者长期使用
- 需要专业的外观
- 需要自动更新功能

### 操作步骤

#### 1. 安装 Node.js
访问 https://nodejs.org/ 下载安装 LTS 版本

#### 2. 初始化 Electron 项目
```bash
# 进入项目目录
cd aphasia-math-recovery

# 初始化 npm 项目
npm init -y

# 安装 Electron
npm install electron --save-dev
```

#### 3. 创建 Electron 入口文件
新建 `main.js`：

```javascript
const { app, BrowserWindow } = require('electron');
const path = require('path');

function createWindow() {
    const mainWindow = new BrowserWindow({
        width: 1200,
        height: 800,
        webPreferences: {
            nodeIntegration: true
        },
        // 应用图标
        icon: path.join(__dirname, 'assets/icon.png')
    });

    // 加载本地文件
    mainWindow.loadFile('index.html');

    // 可选：打开开发者工具
    // mainWindow.webContents.openDevTools();
}

app.whenReady().then(createWindow);

app.on('window-all-closed', () => {
    if (process.platform !== 'darwin') app.quit();
});

app.on('activate', () => {
    if (BrowserWindow.getAllWindows().length === 0) createWindow();
});
```

#### 4. 修改 package.json
```json
{
  "name": "aphasia-math-recovery",
  "version": "1.0.0",
  "main": "main.js",
  "scripts": {
    "start": "electron .",
    "build": "electron-builder"
  },
  "build": {
    "appId": "com.yourcompany.aphasia-math",
    "productName": "算术康复练习",
    "directories": {
      "output": "dist"
    },
    "files": [
      "index.html",
      "src/**/*",
      "main.js"
    ],
    "win": {
      "target": "nsis",
      "icon": "assets/icon.ico"
    },
    "mac": {
      "target": "dmg",
      "icon": "assets/icon.icns"
    }
  }
}
```

#### 5. 打包应用
```bash
# Windows 打包
npm run build

# 输出文件在 dist 目录
# 算术康复练习 Setup 1.0.0.exe
```

#### 6. 分发应用
将生成的 `.exe`（Windows）或 `.dmg`（Mac）文件发送给用户安装

### 优点
- ✅ 专业的外观，像正规软件
- ✅ 双击图标直接打开，无需浏览器
- ✅ 可以添加自动更新功能
- ✅ 可以自定义窗口大小

### 缺点
- ⚠️ 需要学习 Electron
- ⚠️ 打包文件较大（约 100MB+）
- ⚠️ 需要为不同系统分别打包

## 方案三：部署到网站（推荐）

### 适用场景
- 多人使用，随时访问
- 不需要分发文件
- 需要数据统计分析

### 方案 3A：GitHub Pages（免费）

#### 1. 创建 GitHub 仓库
1. 访问 https://github.com/new
2. 仓库名称：`aphasia-math-recovery`
3. 选择 "Public"
4. 点击 "Create repository"

#### 2. 上传代码
```bash
# 在项目目录执行
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/aphasia-math-recovery.git
git push -u origin main
```

#### 3. 启用 GitHub Pages
1. 进入仓库 Settings → Pages
2. Source 选择 "Deploy from a branch"
3. Branch 选择 "main"，文件夹选择 "/ (root)"
4. 点击 Save
5. 等待几分钟，访问显示的网址（如 `https://你的用户名.github.io/aphasia-math-recovery`）

### 方案 3B：Netlify（免费，更简单）

#### 1. 注册 Netlify
访问 https://www.netlify.com/ 用 GitHub 账号登录

#### 2. 拖拽部署
1. 将 `aphasia-math-recovery` 文件夹压缩成 zip
2. 在 Netlify 控制台点击 "Sites" → "Add new site" → "Deploy manually"
3. 拖拽 zip 文件上传
4. 自动获得网址（如 `https://aphasia-math-xxx.netlify.app`）

#### 3. 自定义域名（可选）
1. 购买域名（如阿里云、腾讯云）
2. 在 Netlify 添加自定义域名
3. 配置 DNS 解析

### 优点
- ✅ 无需安装，浏览器直接访问
- ✅ 一次部署，到处使用
- ✅ 自动更新（推送代码后自动部署）
- ✅ 支持 HTTPS，数据安全

### 缺点
- ⚠️ 需要网络连接
- ⚠️ 数据存储在浏览器，换设备需重新练习

## 常见问题

### Q1: 为什么双击 index.html 打不开？
**A:** 检查是否安装了浏览器。如果不行，尝试：
- 右键 index.html → 打开方式 → 选择 Chrome/Edge/Safari
- 确保文件路径中没有特殊字符

### Q2: 数据会保存在哪里？
**A:**
- **方案一/二**：保存在浏览器 LocalStorage，路径：`C:\Users\用户名\AppData\Local\Google\Chrome\User Data\Default\Local Storage`
- **方案三**：保存在浏览器中，清缓存会丢失，建议定期导出备份

### Q3: 如何备份患者数据？
**A:**
1. 在设置页面点击 "导出数据"
2. 保存生成的 JSON 文件
3. 在新设备导入即可恢复

### Q4: 手机能用吗？
**A:**
- **方案一**：可以，将文件传到手机，用浏览器打开
- **方案二**：需要单独打包移动端版本
- **方案三**：可以，浏览器直接访问网址

### Q5: 如何更新版本？
**A:**
- **方案一**：重新复制新版本的文件夹
- **方案二**：重新打包并分发给用户安装
- **方案三**：推送代码到仓库，自动更新
