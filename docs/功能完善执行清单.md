## 🔴 高优先级（1-2周）

### 可视化训练模块（新增）
> 替代原"认知辅助"功能，以专项训练形式满足视觉辅助需求

- [ ] 创建可视化训练入口（基础训练子模块）
- [ ] 加法可视化（🍎图标辅助）
  - [ ] 显示两组物品图标
  - [ ] 用问号表示待求结果
- [ ] 数轴可视化
  - [ ] 显示数字在数轴上的位置
  - [ ] 加法显示向右跳跃箭头
  - [ ] 减法显示向左回退箭头
- [ ] 减法可视化
  - [ ] 显示总数量图标
  - [ ] 划掉需减去的部分
  - [ ] 高亮剩余部分
- [ ] 乘法方阵（点阵表示）

**验收标准**：患者通过可视化训练能建立数字-数量映射关系

### 疲劳检测与休息提醒（新增）
> 防止过度练习导致挫败感，体现产品关怀

- [ ] 疲劳检测逻辑
  - [ ] 连续练习时长检测（>15分钟）
  - [ ] 连续答题准确率检测（最近5题<60%）
  - [ ] 答题速度检测（比平均慢50%）
- [ ] 疲劳等级响应
  - [ ] 轻度：显示鼓励语
  - [ ] 中度：弹出休息建议弹窗
  - [ ] 重度：建议保存进度退出
- [ ] 休息引导动画
  - [ ] 呼吸练习引导
  - [ ] 眼部放松提示
- [ ] 设置中添加开关

**验收标准**：触发条件后正常响应，不影响当前进度，文案温暖友好

---

## 🟡 中优先级（2-4周）

### 智能难度推荐系统（新增）
> 帮助患者选择合适的练习难度

- [ ] 推荐算法
  - [ ] 基于最近10题正确率
  - [ ] >90% 推荐升级
  - [ ] 70-90% 保持当前
  - [ ] <70% 建议降低或复习
- [ ] UI展示
  - [ ] 练习设置页面显示"推荐"标签
  - [ ] 首页"智能练习"按钮使用推荐难度
- [ ] 首次使用后的默认难度

**验收标准**：推荐准确率 > 80%，用户满意度高
