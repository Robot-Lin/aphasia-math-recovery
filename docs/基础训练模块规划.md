# 基础训练模块 产品规划文档

## 一、模块定位
针对命名性失语症患者的**认知基础强化训练**，与现有"智能练习"形成互补：
- 智能练习 → 综合应用、错题针对
- 基础训练 → 肌肉记忆、数感培养

## 二、训练类型规划

### L1 基础层（数感建立）
| 训练项 | 内容 | 康复价值 |
|-------|------|---------|
| 数字认读 | 0-100 数字卡片朗读 | 强化数字-语音映射 |
| 相邻数 | 34 的前面是 __ | 理解数的连续性 |
| 大小比较 | 25 > __ > 23 | 数的大小感知 |
| 奇偶分类 | 区分奇偶数 | 规律识别能力 |

### L2 运算层（计算基础）
| 训练项 | 内容 | 康复价值 |
|-------|------|---------|
| 凑十法 | 8+2, 7+3... | 进位加法基础 |
| 破十法 | 10-7, 15-8... | 退位减法基础 |
| 加法表 | 1+1 到 9+9 | 基础加法肌肉记忆 |
| 乘法表 | 九九乘法表 | 乘法肌肉记忆 |
| 倍数次序 | 2/5/10的倍数 | 规律记忆 |

### L3 应用层（生活数学）
| 训练项 | 内容 | 康复价值 |
|-------|------|---------|
| 时间认读 | 整点、半点 | 生活自理能力 |
| 货币换算 | 1元=10角 | 购物场景训练 |
| 简单测量 | 厘米/米概念 | 空间感知 |

## 三、交互模式设计

### 模式A：闪卡模式（推荐默认）
```
┌─────────────────────────────┐
│                             │
│         [  7 × 8  ]         │
│                             │
│         ─────────           │
│                             │
│           ？                │
│                             │
│    [显示答案]  [下一个]     │
└─────────────────────────────┘
```
- 节奏可控，压力小
- 适合建立自信

### 模式B：填空模式
```
7 × __ = 56

[5] [6] [7] [8]  ← 选项或键盘输入
```
- 双向训练（因→果，果→因）
- 强化等式平衡概念

### 模式C：听算模式
```
🔊 播放："七乘八"

[请作答]
```
- 纯听觉输入
- 训练听觉注意力（失语症常见薄弱环节）

### 模式D：跟读模式
```
"七乘八等于五十六"

[播放] [跟读] [下一个]
```
- 语音+文字双通道
- 语言康复结合数学训练

## 四、难度曲线设计

```
Week 1-2: 数字认读 → 相邻数 → 大小比较
Week 3-4: 凑十法 → 破十法
Week 5-6: 加法表
Week 7-8: 乘法表
Week 9+: 应用层 + 混合训练
```

每类训练内部再分三级：
- Level 1: 1-5范围
- Level 2: 1-9范围
- Level 3: 1-12范围（乘法）

## 五、与现有系统整合

### 数据打通
- 基础训练数据计入总统计
- 错题可加入错题本（尽管是基础题）
- 连续练习天数正常累计

### 入口整合
侧边栏新增：
```
🏠 首页
✏️ 智能练习
📚 基础训练  ← 新增
📖 错题本
📊 历史记录
⚙️ 设置
```

### 首页快捷入口
在现有"快速开始"区域，分两个按钮：
- 【智能练习】→ 进入练习设置
- 【基础训练】→ 直接进入上次训练的下一项

## 六、技术实现要点

### 语音系统
- 使用 Web Speech API 的 SpeechSynthesis
- 预加载语音，避免延迟
- 数字读法优化："56"→"五十六"非"五六"

### 内容配置JSON结构
```json
{
  "category": "multiplication",
  "name": "九九乘法表",
  "levels": 3,
  "modes": ["flashcard", "fillblank", "listen"],
  "items": [
    { "question": "7×8", "answer": 56, "read": "七乘八等于五十六" }
  ]
}
```

### 进度追踪
新增 skillProgress 字段：
```javascript
skillProgress: {
  numberRecognition: 0-100,
  additionFacts: 0-100,
  subtractionFacts: 0-100,
  multiplicationFacts: 0-100,
  divisionFacts: 0-100
}
```

## 七、MVP 优先级

### P0（最小可用）
1. 乘法表闪卡模式
2. 基础训练入口
3. 简单进度记录

### P1（体验完善）
1. 凑十法/破十法
2. 语音朗读
3. 填空模式

### P2（深度优化）
1. 听算模式
2. 能力雷达图
3. 生活数学模块

## 八、预期康复效果

- **4周**：数字反应速度提升
- **8周**：基础运算无需思考（自动化）
- **12周**：可独立完成简单生活计算
