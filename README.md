# 算术康复练习工具

> 专为命名性失语症患者设计的算术康复练习应用

---

## 🎯 项目简介

本工具旨在帮助命名性失语症患者通过循序渐进的算术练习，恢复日常生活所需的计算能力。

### 针对失语症患者的特殊设计

- ✅ **大字体显示** - 题目字号 56px，减少阅读负担
- ✅ **高对比度配色** - 文字与背景对比度 > 4.5:1
- ✅ **超大触摸目标** - 按钮尺寸 80px+，便于操作
- ✅ **即时正向反馈** - 答对时给予视觉和听觉鼓励
- ✅ **无时间压力** - 不显示倒计时，按自己的节奏练习
- ✅ **简洁界面** - 每个页面只做一件事，避免信息过载

## 🎮 使用指南

### 开始练习
1. 点击「开始练习」按钮
2. 系统自动配置：加法 + 初级难度 + 5题
3. 点击「开始练习」进入题目

### 答题方式
- 点击屏幕上的数字键盘
- 或使用电脑键盘输入数字
- 按 Enter 键提交答案
- 按 Backspace 删除最后一位
- 按 Escape 清空答案

### 查看结果
- 答完所有题目后自动显示结果
- 可查看正确率和答对/答错题数
- 错题自动保存，方便后续复习
