# 算术康复练习工具

> 专为命名性失语症患者设计的算术康复练习应用

---

## 🎯 项目简介

本工具旨在帮助命名性失语症患者通过循序渐进的算术练习，恢复日常生活所需的计算能力。

### 针对失语症患者的特殊设计

- ✅ **大字体显示** - 题目字号 56px+，减少阅读负担
- ✅ **高对比度配色** - 文字与背景对比度 > 4.5:1
- ✅ **超大触摸目标** - 按钮尺寸 80px+，便于操作
- ✅ **即时正向反馈** - 答对时给予视觉和听觉鼓励
- ✅ **语音朗读辅助** - 题目和答案自动语音播报
- ✅ **无时间压力** - 不显示倒计时，按自己的节奏练习
- ✅ **简洁界面** - 每个页面只做一件事，避免信息过载

## ✨ 主要功能

| 功能模块 | 状态 | 说明 |
|---------|------|------|
| 🏠 首页 | ✅ | 能力雷达图、今日统计、快速入口 |


## 🎮 使用指南

### 开始练习
1. 从侧边栏选择练习类型
   - 「智能练习」- 自定义运算类型和难度
   - 「基础训练」- 分级难度，循序渐进
   - 「挑战模式」- 动态难度，突破极限
2. 设置练习参数
3. 开始答题

### 答题方式
- **键盘模式**：点击屏幕数字键盘或电脑键盘输入
- **选择模式**：从选项中选择正确答案
- 按 Enter 键提交答案
- 按 Backspace 删除最后一位
- 按 Escape 清空答案

### 挑战模式说明
- 6个难度等级：个位数 → 三位数加法
- 连续答对5题自动升级
- 答错保持当前难度，重置连击
- 记录历史最高等级

### 语音朗读
- 题目自动朗读
- 答错时播报正确答案
- 升级时语音祝贺
- 可在设置中开关

### 查看结果
- 答完所有题目后自动显示结果
- 可查看正确率和答对/答错题数
- 错题自动保存，方便后续复习
